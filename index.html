JailbreakChangelogs Assets

This repository hosts all static assets for the JailbreakChangelogs project.

About This Repository

This is a dedicated repository for storing and serving static assets used by the JailbreakChangelogs rewrite project. All assets are served with long-term caching enabled for optimal performance.

Accessing Assets

Assets can be accessed through the following base URL:

https://assets.jailbreakchangelogs.xyz/assets/

Cache Policy

All assets are served with a cache policy of 1 year (31,536,000 seconds) to ensure optimal performance and reduced server load.
